# Kanboard-QA

This repository is a QA project built around **Kanboard**, an open-source project and task management application.

The project is still **in progress**. I am developing it step by step to practice testing a real product in a more structured way.

At the moment, I am mainly working on:

- understanding Kanboard features and workflows
- preparing test documentation and coverage
- creating manual test cases for the **Project Management** area
- setting up the automation foundation with **Java, Maven, Selenium, and TestNG**

The goal is to grow this repository into an end-to-end QA project that includes analysis, manual testing, UI automation, and API testing.
